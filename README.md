Clone using https://github.com/AnandUnity3d/V18-ManagementApp.git in Visual Studio Code

Navigate to the path till V18-ManagementAppAPI - Node JS project implemented API services
Execute npm i(node packages install) (Please make sure npm installed into system or not)
Run command in terminal - node server.js

Navigate to the path till V18-ManagementFrontEnd - React project implemented for frontend
Execute npm i 
Run "npm run dev"
